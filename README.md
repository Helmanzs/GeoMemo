# 27. Vývoj mobilní aplikace – Uživatelská příručka

Aplikace GeoMemo slouží k uchování drahých vzpomínek z výletů. Po instalaci a první spuštěním aplikace uživatel uvidí domovní obrazovku. Zatím prázdná bez žádných vzpomínek. <br>
<img width="603" height="1305" alt="image" src="https://github.com/user-attachments/assets/524cf2f1-a27b-4f9e-9cec-97548218d7b8" />


V pravém dolním rohu je tlačítko „+“, na které když už uživatel klikne, otevře se stránka, kde si uživatel může uložit novou vzpomínku. Obec se načte automaticky podle polohy uživatele, pokud povolil příslušná oprávnění a má přístup k internetu. Případně může vybrat z předem načteného seznamu některých obcí. Dále může vzpomínku ohodnotit od 1-5, napsat krátký popis a popřípadě vybrat i fotku z galerie. <br> 
<img width="594" height="1200" alt="image" src="https://github.com/user-attachments/assets/58cf52b8-e64d-4fad-bbb7-45b6a7a9f38c" />

Po uložení vzpomínky aplikace uživatele přesměruje zpátky na úvodní stranu, kde nyní vidí svoji přidanou vzpomínku.

Vlevo nahoře se nachází tlačítko na různé druhy seřazení. Uživatel může seřadit vzpomínky podle:
1.	Hodnocení
2.	Poloha (podle blízkosti)
3.	Abecedně
